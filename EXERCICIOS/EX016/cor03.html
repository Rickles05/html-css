As Css

Oque é isso?

CSS (Cascading Style Sheets) é uma linguagem de programação que define a aparência de páginas web. É considerada uma tecnologia fundamental para a criação de websites modernos.

O que faz as CSS?

  • Define como os elementos HTML devem ser estilizados
  • Controlar cores, fontes, tamanhos, layout, margens,
    • preenchimento
  • Separar a apresentação visual do conteúdo HTML
  • Tornar o código mais eficiente e fácil de manter
  • Permitir a criação de designs complexos e responsivos

Como funciona o CSS?

  • É composto por regras de estilo, cada uma contendo um seletor e
    • declarações
  • O seletor indica qual elemento será estilizado, e as declarações
    • definem os estilos
  • A sintaxe CSS consiste em regras de estilo que se aplicam a elementos HTML usando seletores
